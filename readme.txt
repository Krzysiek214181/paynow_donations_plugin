This plugin implements a donation form for paynow payments, an admin dashboard with transaction history, notification debug menu and settings

== Description ==
for full description go to https://github.com/Krzysiek214181/donations_for_paynow_plugin

== Installation ==
To set up the plugin you have to be the owner of an active paynow account

Simply copy the ApiKey and SignatureKey from your paynow panel into the corresponding field in the Paynow -> Paynow Settings menu and save them.

Set the Environment setting to PRODUCTION

In your paynow panel set up the correct Shop Domain

Set the Notifications URL to https://yourdomain.example/wp-json/paynownotifications/notify

Set the Return address to a site where you\'ll use the [paynow_return] shortcode

For testing setup go to https://github.com/Krzysiek214181/donations_for_paynow_plugin
